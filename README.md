# Dark Light Portfolio UI

A modern portfolio landing page with dark and light mode built using HTML, CSS, and JavaScript.

## Features

- Responsive Navigation
- Modern Hero Section
- Dark / Light Mode Toggle
- Dynamic Icon Switching
- Minimal Portfolio Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap Icons

## JavaScript Concepts Used

- DOM Manipulation
- Event Listeners
- classList.toggle()
- textContent
- Dynamic UI Updates
